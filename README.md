# Cartmation 

A Netflix-inspired streaming platform built with Django.

## Features
- 🎬 Movies, TV Series & Anime library
- 🔍 Live search with suggestions
- ⭐ Star ratings system
- 👍 Likes & dislikes
- 💬 Comments & reviews
- ▶️ Video player with YouTube embed
- 🏆 Top 10 leaderboard
- 🎭 Actor spotlight collections
- 👤 User profiles & watchlist
- ⚙️ Account settings
- 🎨 Auto-playing hero slideshow
- 📱 Responsive design

## Setup

`ash
# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Create superuser
python manage.py createsuperuser

# Run server
python manage.py runserver
`

## Tech Stack
- **Backend:** Django 4.2
- **Database:** SQLite
- **Frontend:** Bootstrap 5, Font Awesome
- **Storage:** Local file system

## Admin
Visit /admin/ to manage content.
